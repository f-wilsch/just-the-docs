Coupling

Coupling[label,{indices},order]

denotes the coupling label with a set of indices and a given order in the EFT expansion.

Details and Options

  • A coupling can be obtained with the short-cut label[indices] after it was defined using DefineCoupling[label,Indices->{indices},EFTOrder->order].
