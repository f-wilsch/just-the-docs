CovariantLoop

CovariantLoop[lagrangian,{field1, field2, …}]

calculates the one-loop supertraces involving exactly the degrees of freedom specified in the field list.

Details and Options

  • The following options can be given:
  • EFTOrder6specifies the targeted order in the EFT power counting.
    ModeMatchingsets the kind of loop calculation performed.
  • The values of the Mode option are
  • Matchingthe function outputs the EFT matching contribution resulting from the trace(s).
    Divergencethe function returns the 1/ϵ pole of the trace(s). N.b. it is the divergence of a contribution to the effective action and minus the value of the counterterms needed to renormalize the theory.
