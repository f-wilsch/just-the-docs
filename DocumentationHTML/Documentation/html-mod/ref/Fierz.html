Fierz

Fierz[SP$1 * SP$2]

returns the fierzing of the product of SP1 and SP2 by exchanging the second spinor in each closed spin chain.

Details and Options

  • The following options can be given:
  • Order{1, 4, 3, 2}Order[expr1,expr2] gives 1 if expr1 is before expr2 in canonical order, and -1 if expr1 is after expr2 in canonical order. It gives 0 if expr1 is identical to expr2.
    EvanescentTrueEvanescent is an option for Fierz, GammaReduction and RemoveLCTensor. If set to True (default), it stores the evanescent operator (if generated) in an association and includes it in the result.
