Matchete`

γ

γ[index]

is short-hand input for a Dirac matrix γμ with index as Lorentz label.

γ[5]

is short-hand input for the γ5 matrix.

γ[index1, index2, …]

is short-hand input for an anti-symmetrized product of Dirac matrices Γμν… with index1, index2, … as Lorentz labels.

Details

  • Internally the Dirac matrix objects are represented as GammaM and Gamma5.

Examples

Basic Examples  (1)

Any input with γ is immediately converted to internal Matchete representation. They nevertheless display conveniently to the user with NiceForm.
