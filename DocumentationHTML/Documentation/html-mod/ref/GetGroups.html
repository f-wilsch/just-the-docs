GetGroups

GetGroups[]

returns an association of all the groups that have been defined by DefineCGGroup, DefineGaugeGroup and DefineGlobalGroup.

Details and Options

  • The association contains the group names and the Lie algebras of each group as Alg objects.

Examples

Basic Examples  (1)

After initializing a group it appears when calling GetGroups.
