MapEffectiveCouplings

MapEffectiveCouplings[L$0,LTarget]

attempts to relate the two input Lagrangians, expressing the effective couplings in LTarget through the matching coefficients in L0. The option SortByEFTOrder (default is True) specifies whether the output is sorted by power-counting.

Details and Options

  • The following options can be given:
  • SortByEFTOrderTrueSortByEFTOrder is an option for MapEffectiveCouplings, specifying whether the output rules are sorted by power-counting. The default is True, but setting it to False can lead to minor speed-up for very large Lagrangians.
