PrintEffectiveCouplings

PrintEffectiveCouplings[L]

prints a list of effective couplings defined by EOMSimplify appearing in the input Lagrangian L, along with their definitions. With the option Rules True, the function will return the replacement list for these couplings, instead.

Details and Options

  • The following options can be given:
  • RulesFalseRules is an option to specify whether output should be a list of replacement rules instead of printed.
