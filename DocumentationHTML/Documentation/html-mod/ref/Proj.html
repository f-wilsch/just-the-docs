Matchete`

Proj

Proj[+1]

is the internal representation for the right-handed projection matrix PR.

Proj[-1]

is the internal representation for the left-handed projection matrix PL.

Details and Options

  • The internal representation of the projection matrices is rarely used directly by user; they can be more conveniently entered using the PR and PL short-hand.
  • Note that other Matchete manipulations assume that all Dirac matrices come wrapped inside the DiracProduct head.
