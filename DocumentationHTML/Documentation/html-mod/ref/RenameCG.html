RenameCG

RenameCG[oldName newName]

assigns a new name (reference symbol) to a CG that is already initialized.

Details and Options

  • After renaming a CG the old CG objects CG[oldName, {…}] are no longer valid expressions.
  • N.b. Renaming a CG in the internal Matchete state does not change any CGs that occur in expressions (e.g. Lagrangians). Such expressions are no longer valid after the renaming.
