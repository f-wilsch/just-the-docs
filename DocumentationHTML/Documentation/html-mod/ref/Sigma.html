σ

σ[mu, nu]

is short-hand input for the Hermitian, anti-symmetric Dirac matrix σμν with indices mu,nu as Lorentz labels.

Details

  • σ does not exist an internal object in Matchete. It is represented as a GammaM object using the relation .

Examples

Basic Examples  (1)

Any input with σ is immediately converted to internal Matchete representation. It nevertheless displays conveniently to the user with NiceForm.
