Matchete

    by Javier Fuentes-Martín, Matthias König, Julie Pagès, Anders Eller Thomsen, and Felix Wilsch

Matchete is a package to perform EFT matching, RG calculations, and various manipulations of Lagrangians.

Define a Model

DefineGaugeGroup — define a new gauge group that can be used for fields and in Lagrangians

DefineField — define a new field that can be used in Lagrangians

DefineCoupling — define a new coupling that can be used in Lagrangians

LoadModel — load a predefined model from the model directory

DefineFlavorIndex  ▪  DefineGlobalGroup

RG calculations

Match — integrate out heavy fields from a Lagrangian and obtain the EFT

Lagrangian simplifications

GreensSimplify — simplifies a Lagrangian to an off-shell basis

EOMSimplify — simplifies a Lagrangian to an on-shell basis

MapEffectiveCouplings — determine EFT of target basis based from a matched EFT
