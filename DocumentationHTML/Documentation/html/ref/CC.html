CC

CC

is short-hand input for the charge conjugation matrix C.

Details and Options

  • Internally the charge conjugation matrix is represented as GammaCC.

Examples

Basic Examples  (1)

Any input with CC is immediately converted to internal Matchete representation. They nevertheless display conveniently to the user with NiceForm.
