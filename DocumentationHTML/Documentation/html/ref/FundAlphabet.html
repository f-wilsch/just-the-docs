FundAlphabet

FundAlphabet

is an option for the routine DefineGaugeGroup that specifies a printing alphabet for the indices of the fundamental representation.

Details and Options

  • The canonic indices of the fundamental representation that are obtained by invoking functions like RelabelIndices print with the labels specified in FundAlphabet when using NiceForm.
