GammaCC

GammaCC

is the internal representation for the charge conjugation matrix C.

Details and Options

  • The internal representation of the charge conjugation matrix is rarely used directly by user; it can be more conveniently entered using the CC short-hand.
  • Note that other Matchete manipulations assume that all Dirac matrices come wrapped inside the DiracProduct head.
