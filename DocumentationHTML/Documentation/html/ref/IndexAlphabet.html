IndexAlphabet

IndexAlphabet

is an option for the routine DefineRepresentation and DefineFlavorIndex that specifies a printing alphabet for the representation or flavor indices.

Details and Options

  • The canonic indices of the specified representation or of the flavor indices that are obtained by invoking functions like RelabelIndices print with the labels specified in IndexAlphabet when using NiceForm.
